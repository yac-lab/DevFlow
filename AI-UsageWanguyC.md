### Zouti AI #1

*Non/Platfòm Zouti a:* Perplexity AI  
(ekz., ChatGPT, Grammarly, GitHub Copilot, Rechèch Google, elatriyeee.)

*Dat & Lè Itilizasyon an:* 14/02/2026

*Objektif Itilizasyon an:* Asistans nan chwa achitekti (dependencies) ak debogaj lojik API. 
(Ekz., Deboge kòd, Asistans rechèch, elatriyee.)

*Prompt oubyen Rekèt ki te fèt la*

["Quelles sont les meilleures dépendances Flutter pour un projet de Flashcards avec API et persistance ? Comment corriger une erreur de chargement JSON depuis un lien Gist GitHub ?"]






*Tout repons/output zouti AI a te bay la:*

[•  Lis depandans: http, shared_preferences, provider, url_launcher, google_fonts.
•  Konsèy teknik: Sèvi ak /raw nan fen lyen Gist la pou jwenn done JSON yo dirèkteman.
•  Kòd FIX pou progress bar la: value: provider.progress.clamp(0.0, 1.0).
]




*Kòman w te itilize repons ou te jwenn nan:*  
(Mwen te entegre depandans yo nan pubspec.yaml. Mwen te modifye _baseUrl nan ApiService pou ajoute /raw paske lien an pa t ap mache. Mwen te aplike metòd .clamp() sou progress bar la pou evite erè vizyèl lè valè a depase 1.0.)







*Kisa w aprann sou repons ou jwenn nan?*  
(Mwen aprann diferans ant yon lyen koòdone GitHub ak yon lyen raw pou API. Mwen aprann tou enpòtans pou "limiter" (clamp) valè nimerik nan Flutter pou UI a pa fè aksidan)

### Zouti AI #2
Rekopye menm seksyon anlè a, si gen lòt zouti

---Non/Platfòm Zouti a : Gemini (Google)

Dat & Lè Itilizasyon an : 16/02/2026

Objektif Itilizasyon an : Rezolisyon konfli vèsyon sou Git ak kowòdinasyon workflow ekip la.

Prompt oubyen Rekèt ki te fèt la : "Comment résoudre une erreur [rejected] (non-fast-forward) sur VS Code alors que mes collègues travaillent sur des branches différentes (master/main) ?"

Tout repons/output zouti AI a te bay la : Enstriksyon pou fè git pull origin master anvan git push epi kijan pou verifye branch aktif la ak git branch.

Kòman w te itilize repons ou te jwenn nan : Mwen itilize kòmandman yo nan tèminal VS Code pou m ka uploade kòd final la sou GitHub san pèdi travay kòlèg mwen yo.

Kisa w aprann sou repons ou jwenn nan? Mwen aprann kijan pou m jere yon workflow kolaboratif ak Git, espesyalman enpòtans pou senkronize depo lokal la ak depo a distans anvan nenpòt modifikasyon enpòtan.
